# Part 4: AI Solution Design - Intelligent Support Ticket Routing

## Project Overview
This project identifies a critical business bottleneck in the Customer Support domain—high manual processing hours—and proposes a data-driven AI solution. By leveraging Natural Language Processing (NLP), we aim to automate ticket categorization to improve efficiency and customer satisfaction.

## Business Problem
Based on the provided KPI data, the current manual process requires over 400+ hours monthly for ticket sorting, leading to an average resolution time of approximately 30 hours. This delays response times and impacts the overall customer experience.

## Proposed Solution
An **LSTM-based Text Classification model** that automatically reads customer queries and routes them to the appropriate department.
